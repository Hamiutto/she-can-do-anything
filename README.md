# 她行 She Can Do Anything

> 35岁？结婚了吗？有孩子吗？这些问题不该是枷锁，而该是我们走过来之后胸前闪闪发亮的勋章。  
> 她行是一个基于「五种时间」理论的女性互助社区：真实的人生样本、灵感粒子问答、回声感谢信，让每一个正在经历困境的女性，找到那个已经走过去的姐妹。  
> 我们全员选择「好玩时间」赛道，因为当一件事变得好玩时，你自然就能坚持下去。

越是 AI 信息冲击的时代，人和人之间真实的交流就越显珍贵。

她行面向真实女性经验，构建了一个不卖课、不打鸡血、不制造焦虑的互助问答社区。我们只想做一件事：让女性找到同频者，互相照亮。

## 项目亮点

- 真实的人生样本库，连接真实女性的经历、故事和标签
- 五种时间体系，把人生阶段变成可以被理解和选择的路径
- 问问她的粒子问答，把“提问”和“回答”变成更有情绪记忆的体验
- 隐私保护，游客可自由浏览，只有在需要沉淀个人内容时才触发登录注册
- 前后端已打通，可直接本地运行体验完整流程

## 核心功能

### 1. 人生样本库

21 位真实评委 + 持续增长的用户样本卡。每个人物都包含完整故事、标签、五种时间分类。

用户可以通过城市、行业、MBTI、星座、爱好等多维度搜索，找到“世界上另一个她”。

### 2. 五种时间体系

生存 / 赚钱 / 好看 / 好玩 / 心流。

每种时间下都有对应的人生模板和真实故事。用户可以找到自己正处的时间阶段，看到同类人的活法。

### 3. 问问她

独创“粒子漂流瓶”交互设计。每个飘动的粒子都是一个等待被回答的问题。

用户可以公开提问、回答问题。感谢信（回声瓶）会作为下一步继续产品化，形成“提问 → 回答 → 感谢 → 再触发”的情感闭环。

### 4. 隐私保护

游客可以自由浏览全部内容。仅在提问、回答、保存个人资料等需要沉淀个人内容的动作时触发登录注册。

所有用户数据都在本地后端中安全处理，不向第三方泄露个人故事。

## 推荐体验路径

1. 打开首页 `/`
2. 在 `She is ____` 区域输入关键词，或点击标签进入人物样本
3. 进入 `/life-templates` 查看匹配出来的人生样本卡
4. 打开 `/qa-square`，点击“公开提问”提交一个问题
5. 点击“回答问题”进入 `/particle`，在粒子画布中点击问题并回答
6. 打开 `/login` 注册账号，再进入 `/profile` 完善“我的人生样本卡”

## 本地运行

需要同时启动后端和前端。如果已经安装过依赖，可以跳过 `npm install`。

### 1. 启动后端

```powershell
Set-Location backend
npm install
npm run dev
```

后端默认运行在 `http://localhost:3001`。

### 2. 启动前端

```powershell
npm install
npm run dev
```

前端默认运行在 `http://localhost:8887`。

## 技术架构

| 模块 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue 3 + Vite + Vue Router | 单页应用与页面路由 |
| 后端 | Node.js HTTP server | 提供认证、资料、问答、通知 API |
| 数据 | JSON + `questions.db` | 本地保存用户、问题和回答数据 |
| 认证 | JWT | 登录态恢复与 API 鉴权 |
| 交互 | Canvas + 粒子系统 | 问题粒子、鼠标跟随、点击回答 |
